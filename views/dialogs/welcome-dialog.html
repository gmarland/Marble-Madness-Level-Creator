Welcome to Marbles 3D Web Painter

In order to paint click left mouse button. If you want to erase use the right mouse button. You can also look around with the middle mouse button.

You can move with [W][A][S][D][R][F] and control the grid using the arrow keys. You can find the full list by clicking "Controls" in the settings menu.

The code is available if you are interested on GitHub at https://github.com/gmarland/Marbles-3D-Web-Painter. If you have some thoughts then please let me know @garethmarland
